== Install git

* Linux

[source,bash]
----
[pm] install git
----

``pm`` - package manager (``apt``, ``yum``, ``dnf``, etc)

CAUTION: Ensure that manager has ``install`` option (e.g. ``apk`` uses ``add`` instead of ``install``)
